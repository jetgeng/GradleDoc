  本程序是一个Google Appengine的第三方上传工具，不需要配置Python和GAE SDK的
运行环境，因为它是一个weindow的命令行程序。

    操作：在window命令行输入操作命令。

上传命令：SDUpload update sdblog
    其中：SDUpload为程序名；update为更新或上传的命令；sdblog是程序所在目录。



    更多方法请参考http://www.quke.cn
